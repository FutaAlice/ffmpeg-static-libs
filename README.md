# ffmepg-static-libs

FFmpeg static libraries built with VS2015 for Windows development.

## Download

See [releases](https://github.com/FutaAlice/ffmpeg-static-libs/releases) page.

## Directory structure

```bash
./ffmepg-x.x.x
├── bin
│   ├── x64
│   │   ├── ffmpeg.exe
│   │   ├── ffplay.exe
│   │   └── ffprobe.exe
│   └── x86
│       ├── ffmpeg.exe
│       ├── ffplay.exe
│       └── ffprobe.exe
├── include
│   ├── libavcodec/...
│   ├── libavdevice/...
│   ├── libavfilter/...
│   ├── libavformat/...
│   ├── libavutil/...
│   ├── libpostproc/...
│   ├── libswresample/...
│   └── libswscale/...
├── lib
│   ├── x64
│   │   ├── libavcodec.lib
│   │   ├── libavcodecd.lib
│   │   ├── libavdevice.lib
│   │   ├── libavdeviced.lib
│   │   ├── libavfilter.lib
│   │   ├── libavfilterd.lib
│   │   ├── libavformat.lib
│   │   ├── libavformatd.lib
│   │   ├── libavutil.lib
│   │   ├── libavutild.lib
│   │   ├── libpostproc.lib
│   │   ├── libpostprocd.lib
│   │   ├── libswresample.lib
│   │   ├── libswresampled.lib
│   │   ├── libswscale.lib
│   │   └── libswscaled.lib
│   └── x86
│       ├── libavcodec.lib
│       ├── libavcodecd.lib
│       ├── libavdevice.lib
│       ├── libavdeviced.lib
│       ├── libavfilter.lib
│       ├── libavfilterd.lib
│       ├── libavformat.lib
│       ├── libavformatd.lib
│       ├── libavutil.lib
│       ├── libavutild.lib
│       ├── libpostproc.lib
│       ├── libpostprocd.lib
│       ├── libswresample.lib
│       ├── libswresampled.lib
│       ├── libswscale.lib
│       └── libswscaled.lib
└── licenses/...
```
